# Welcome to Underpost

Underpost is a comprehensive framework for building Progressive Web Apps (PWA) and API-REST projects with full CI/CD capabilities. Develop, build, deploy, test, monitor, and manage multiple runtime applications on virtual machines or container instances.

## Technology Stack

<Note>
Underpost is built on modern, production-ready technologies:
- **Node.js v24.10.0** - JavaScript runtime
- **npm v11.6.2** - Package manager
- **MongoDB v7.0** - NoSQL database
- **Rocky Linux v9.6** - Production OS environment
</Note>

## What You Can Build

Underpost provides a complete foundation for:

- **Progressive Web Applications** - Build offline-capable, installable web apps
- **REST APIs** - Create scalable backend services with Express
- **Microservices** - Deploy federated application architectures
- **Full-Stack Applications** - Combine client and server in unified projects

## Next Steps

<Steps>
  <Step title="Install Underpost">
    Install the framework globally via npm to access the CLI tools
    ```bash
    npm install -g underpost
    ```
  </Step>
  <Step title="Create Your Project">
    Generate a new project with the scaffolding command
    ```bash
    underpost new my-app
    ```
  </Step>
  <Step title="Start Building">
    Your development server will be running at http://localhost:4001
  </Step>
</Steps>
